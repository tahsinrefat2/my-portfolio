PostureAware

Multimodal Deep Learning & IoT

Project Overview

Developed a multimodal deep learning model combining CNN-LSTM for time-series sensor data and MLP for user metadata to predict pain intensity, frequency, and location among garment workers. The system uses wearable sensors to monitor posture and movement patterns.

Technical Details

  • CNN layers for spatial feature extraction
  • LSTM layers for temporal pattern recognition
  • MLP branch for demographic metadata
  • Fusion layer for multimodal inputs

Results

Achieved 98% accuracy in pain intensity prediction and 99% in pain location classification.
